# Heart Disease Analysis Project
This project demonstrates data analysis, visualization, 
and machine learning using a Heart Disease dataset.

## Project Structure
## Project Goals
* Data loading and overview
* Data cleaning and validation
* Exploratory data analysis (EDA)
* Correlation analysis
* Building machine learning model (Random Forest)
* Handling class imbalance with SMOTE
* Evaluating model performance

## Visualizations
* Age distribution
* Target distribution (healthy vs disease)
* Age vs Heart Disease (boxplot)
* Correlation heatmap
* Top 10 features correlated with target
* Age vs Cholesterol (scatterplot)
* Cholesterol vs Heart Disease (boxplot)
* Max Heart Rate vs Heart Disease (boxplot)
* Feature Importance
* Confusion Matrix

## Key Findings
* Final model accuracy: 80%
* Most important features: exang, oldpeak, trestbps
* Dataset was imbalanced — fixed with SMOTE
* Recall for healthy patients improved from 0.12 to 0.47
